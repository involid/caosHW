Как запускать тесты?

Даём права на исполнение скрипту
chmod +x run.sh

Называем файл с задачей connect_n_processes.c
И помещаем в текущую директорию

Из текущей директории запускаем скрипт
./run.sh
Смотрим на результат работы

Комментарий:
Скрипт собирает программы на C++ в директории CAOS_test
Скрипт собирает текущую программу
Скрипт добавляет в стандартный путь PATH для поиска программ директорию CAOS_test
Последнее изменение локально, то есть не сохранится при выходе из скрипта
По крайней мере не должно, хотя это не создаст проблемы в любом случае
После чего скрипт запускает программу

Можно тот же самый запуск сделать средствами bash с помощью |
Для этого переходим в директорию CAOS_test и запускаем скрипты
./make.sh
./test_run.sh
./clear.sh
