Ожидаемое имя программы:
decrypt.c
Тест запускается скриптом:
run.sh
Создаваемые в процессе теста файлы с зашифрованным и расшифрованным текстом
чистятся с помощью:
clear.sh

Есть четыре основных скрипта, которые используются при тестировании:
1) encrypt_sh.sh
Зашифровать текст с помощью команды openssl для терминала
2) encrypt_c.sh
Зашифровать текст с помощью программы на C, использующей openssl
Её код во многом похож на решение задачи
3) decrypt_sh.sh
Расшифровать текст с помощью команды openssl для терминала
4) decrypt_c.sh
Расшифровать текст с помощью программы на C, которая и будет решением задачи

Оба скрипта для шифрования эквивалентны, то есть дают одинаковый результат
В случае корректно написаннной программы оба скрипта для
дешифрования тоже будут давать одинаковый результат

Исходный текст находится в файле text.txt
Текст, к которому применили шифрование, будет находиться в файле text_encrypt.txt
Текст после дешифрования будет находиться в файле text_decrypt.txt
